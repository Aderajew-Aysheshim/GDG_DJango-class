# OOP Todo App (CLI) with JSON Persistence

A simple yet robust command-line Todo application built using Object-Oriented Programming principles in Python. Tasks are persistently stored in a local `todos.json` file.

## Features

- Add new tasks
- View all tasks with ID and status
- Update task title
- Toggle task completion
- Delete tasks
- Data persists between sessions using JSON

## How to Run

1. Make sure you have Python 3.6+ installed.
2. Save the code as `todo.py`
3. Run the application:

```bash
python todo.py
